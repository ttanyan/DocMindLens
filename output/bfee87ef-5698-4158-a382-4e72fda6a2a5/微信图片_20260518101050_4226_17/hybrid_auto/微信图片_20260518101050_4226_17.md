# 案例 24. 减轻处罚——未满 16 周岁

# 一、基本案情

2018年8月×日，当事人李某持《港澳居民来往内地通行证》从皇岗口岸入境大厅入境，未向海关书面申报，经检查，在当事人随身携带的行李内发现有化妆品14件，被查获。

# 二、案件办理分析

# （一）定性判断。

首先，当事人携带应当申报物品进境未申股，其行为已经违反海关规定，属于违法。其次，从查获情况看，本案当事人携带方式属于正常携带，在现有证据材料的基础上，没有具体行为情节及证据可以证明当事人存在故意诚盗、骗私等逃避海关监管的主观意图；同时没有被海关征税、退运、移送缉私的记录，综合以上，依据法律法规规定，当事人的行为应当定性为违规。

认定依据：

《处罚实施条例》第十二条规定“违反海关法及其他有关法律、行政法规和规章但不构成走私行为的，是违反海关监管规定的行为”。

走私行为的判定：

1.《海关法》第八十二条规定“有下列情形之一的，无走私行为：（一）运输、携带、邮寄国家禁止或者限制进出口货物、物品或者依法应当缴纳税款的货物、物品进出废的；……”。

2.《处罚实施条例》第七条(二)项规定“经过设立海关网

地点，以液匯、伪装、瞒报、伪报或者其他方式逃避海关监管，运输、携带、邮寄国家禁止或者限制进出境的货物、物品或者依法应当缴纳税款的货物、物品进出境的”。

3.《两简案件操作办法》第十九条规定“当事人运输、携带、邮寄禁止或者限制进出境的货物物品或者应税货物物品进出境，有下列行为之一的，定性走私行为：（一）当事人以藏藻、伪装等方式逃避海关监管的……

# （二）案值认定。

本案中，在扣涉案标的物共14件，且无相同物品，应当以“物品”计核税款。根据当事人提供的购买凭证，结合违法物品的数量及完税价格，进而计算出案值及税款。经计核，14件物品共计0.77万元，税款0.15万元。因此，本案案值0.92万元。

判断依据：

1.“货物”、“物品”的区分，根据《处罚实施条例》第六十四条规定“‘物品’，指个人以运输、携带等方式进出境的行李物品、邮寄进出境的物品，包括货币、金银等。超出自用、合理数量的，视为货物”。

2.《快办系统应用配套制度》“涉案货物、物品区分原则”规定“同类物品数量明显不合理（超过20件）的视为货物”

3.《计核走私偷逃税款办法》第十六条规定“涉嫌走私的货物能够确定成交价格的，其计税价格应当以该货物的成交价格为基础审核确定”。
